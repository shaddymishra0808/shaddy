Create an ultra-fast, highly secure Discord bot using Node.js and discord.js v14 with an advanced hacker-style terminal control panel that shows live logs for every action and allows full control directly from the terminal.

The bot must feel like a real hacker console tool with instant response, real-time execution logs, progress indicators, and a clean command menu system.

The project must be production-ready and optimized to run on AWS EC2, VPS, Replit, PM2, and local machines.

=====================================

CORE TECHNOLOGY REQUIREMENTS

Use the following technologies:

Node.js
discord.js v14
fs
readline
chalk
ora
cli-progress
dotenv

Coding requirements:

• Use async/await everywhere
• Use Promise.all batching for speed
• Use Map for command handlers
• Load JSON files once into memory
• Use caching for performance
• Response execution must start under 150ms

=====================================

CONFIGURATION SYSTEM

Create config.js:

module.exports = {
prefix: "+"
};

Create environment configuration using dotenv.

.env

BOT_TOKEN=
CLIENT_ID=
OWNER_ID=
PREFIX=+

Create ".env.example" for deployment.

The bot must load configuration dynamically.
